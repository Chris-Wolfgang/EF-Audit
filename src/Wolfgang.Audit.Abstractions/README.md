# Wolfgang.Audit.Abstractions

Shared contracts for the `Wolfgang.Audit.*` family of EF audit/change-tracking libraries.

Contains the interfaces, attributes, and entity types consumed by `Wolfgang.Audit.EFCore` (and the future `Wolfgang.Audit.EFLegacy`). No EF Core dependency — pure POCOs and interfaces.

See the [project README](https://github.com/Chris-Wolfgang/EF-Audit) for the full picture.
