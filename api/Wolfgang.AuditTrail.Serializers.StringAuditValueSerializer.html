Remarks

Encoding rules:

  • String stored as-is.
  • Integral, decimal, and floating-point types use InvariantCulture.
  • DateTime and DateTimeOffset use the round-trip "o" format.
  • Guid uses the "D" format (32 digits separated by hyphens).
  • bool renders as True / False.
  • byte[] is base64-encoded.
  • Enums use ToString() (the symbolic name); the discriminator records the enum's full type name.
  • null writes SQL NULL; the discriminator still records the declared CLR type.
