Classes

AuditValueSerializerContractTests<TSut>

Contract test base every IAuditValueSerializer implementation should inherit. Mixes FsCheck-generated property tests (~100 random inputs per type per run) with explicit boundary theories that document known edge cases.

InMemoryAuditValueBuffer

In-memory IAuditValueWriter / IAuditValueReader used by the contract tests. Stores the most recent value written to each column and serves it back on read.

Enums

AuditValueSerializerContractTests<TSut>.SampleEnum

A small enum used exclusively by the contract tests so consumers inheriting this base get coverage of the Enum:<FullName> discriminator path. The Enum suffix is intentional — the type name appears in tests that read "Sample enum value Beta" and the suffix keeps that wording precise.
