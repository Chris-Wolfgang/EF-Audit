Serializes an entity's primary key value(s) into the string stored in AuditHeader.EntityKey.

public interface IAuditEntityKeySerializer

Remarks

Composite keys are passed as a multi-element sequence; single-column keys arrive as a sequence of one element. The default implementation joins parts with '|'; consumers can swap to JSON or any other format.
