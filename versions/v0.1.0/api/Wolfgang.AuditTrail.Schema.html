Wolfgang.AuditTrail

Namespace Wolfgang.AuditTrail.Schema

Classes

AuditMigrationsDbContext

Library-internal DbContext whose model contains only the audit entities — AuditHeader, AuditDetail, and the version tracking row AuditSchemaVersion. Used by AuditSchemaMigrator to install or upgrade the audit schema without dragging the consumer's own entity model into the diff.

AuditSchemaConstants

Compile-time constants shared across the schema-migration components.

AuditSchemaInstaller

Creates and drops the audit tables in the consumer's database.

AuditSchemaVersion

One-row record tracking the audit schema version currently installed in the target database. The row is upserted by AuditSchemaMigrator after a successful migration apply.
