Remarks

UserId is the principal whose credentials authenticated the operation — typically the application's service account or the directly logged-in user.

OnBehalfOfUserId is populated in web/service scenarios where the authenticated principal is acting on behalf of a real end user. Example: an ASP.NET Core application runs as svc-orders (the service account => UserId) but handles a request from steve (the human => OnBehalfOfUserId).
