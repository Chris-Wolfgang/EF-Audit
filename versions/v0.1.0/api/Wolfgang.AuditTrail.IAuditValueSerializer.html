Encodes values captured by the audit interceptor into the column(s) it persists on the AuditDetail row, and decodes them back on read.

public interface IAuditValueSerializer

Remarks

The serializer declares the value column(s) it expects on the AuditDetail table via Columns. v1's DDL path is just Database.EnsureCreatedAsync against the EF Core model — which contains the fixed ValueText / ValueBinary property set on AuditDetail regardless of which serializer is configured. Columns is therefore advisory in v1 (used by the serializer to keep its writes consistent with the entity properties); a future schema installer that emits per-serializer DDL is planned but not shipped.

v1 ships StringAuditValueSerializer (text-only). Future implementations (binary, hybrid) are additive — they slot in without changing this interface.
